SequenceStatCollector Users' Guide
-----------------------------------

1. Functionality
-----------------
This program will output the statistics regarding the
presence of duplicates in the input FASTA sequence file.

2. How to run
--------------
  -- Open a command line window and change directory to where you have SequenceStatCollector.exe
  -- Use the following command with proper arguments to run,
      
	   SequenceStatCollector.exe /fastFile=PathToYourFASTAFile /outDir=DirectoryPathToOutputStats
